## Description
nestjs toy project

## Installation

```bash
# generate a dotenv file
$ touch .env
# and then fill in the required data shown in .env.example 

# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode (using nest)
$ npm run start:dev

# watch mode (using nodemon)
$ npm run start:demon
```

## View API via Swagger

go to ```localhost:APP_PORT/api```
